# IDEA 手册

## 批量替换大小写

例如需要将下面文件中的 `Bxx`、`Cxx` 改成首字母小写可以使用 `( +)(\w)` 匹配到 `  B` 和 `    C`，使用 `$1\l$2` 执行替换。其中 `$1` 代表 `( +)` 所搜索到的部分，`$2` 代表 `(\w)` 搜索到的部分，`\l` 表示将 `$2` 执行小写操作

```
Axx: 123
  Bxx: 123
    Cxx: 123
```

::: tip

`\l` 为替换小写，直到下一个字符；`\u` 为替换大写，直到下一个字符；`\L` 为替换小写，直到字符结束；`\U` 为替换大写，直到字符结束；

:::
