## Problem

Source: [LeetCode 5](https://leetcode-cn.com/problems/longest-palindromic-substring/){target="_blank"}

### Description

给你一个字符串 `s`，找到 `s` 中最长的回文子串。

示例 1：

```text
输入: "babad"
输出: "bab"
注意: "aba" 也是一个有效答案。
```

示例 2：

```text
输入: "cbbd"
输出: "bb"
```

示例 3：

```text
输入：s = "a"
输出："a"
```

示例 4：

```text
输入：s = "ac"
输出："a"
```

提示：

- `1 <= s.length <= 1000`
- `s` 仅由数字和英文字母（大写和/或小写）组成
