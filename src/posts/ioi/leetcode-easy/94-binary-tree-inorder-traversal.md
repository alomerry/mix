Source: [LeetCode 94](https://leetcode-cn.com/problems/binary-tree-inorder-traversal/){target="_blank"}

## Description

给定一个二叉树的根节点 `root` ，返回它的 **中序** 遍历。

示例 1：

```text
输入：root = [1,null,2,3]
输出：[1,3,2]
```

示例 2：

```text
输入：root = []
输出：[]
```

示例 3：

```text
输入：root = [1]
输出：[1]
```

示例 4：

```text
输入：root = [1,2]
输出：[2,1]
```

示例 5：

```text
输入：root = [1,null,2]
输出：[1,2]
```

提示：

- 树中节点数目在范围 `[0, 100]` 内
- `-100 <= Node.val <= 100`


进阶：你可以设计一个时间复杂度为 O(log n) 的解决方案吗？
